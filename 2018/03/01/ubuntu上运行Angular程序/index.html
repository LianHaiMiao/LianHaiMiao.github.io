ubuntu上运行Angular程序

Posted on 2018-03-01 | In 编程 , 前端

ubuntu 16.04 上启动 angular 程序的记录。

1、先装 npm 和 node

不推荐使用 apt-get install 因为版本可能比较老。所以直接上官网下载。

Node.js 和 npm 找到对应版本，直接下载

下载完成之后，进行解压，笔者将解压后的文件放在了 /home/ubuntu/software 目录下。

为了能够在命令行中直接运行 npm 和 node 此处会采取软链接的方式，连接到 /user/local/bin 下。

ln -s /home/ubuntu/software/node-v8.9.4-linux-x64/bin/node /user/local/bin/node

ln -s /home/ubuntu/software/node-v8.9.4-linux-x64/bin/npm /user/local/bin/npm

然后，进行测试。

npm -v

node -v

显示出版本信息就表示 node 安装成功。

2、安装 cnpm

npm 有点慢… 所以我们直接整 cnpm

两种方法：

1、cnpm (gzip 压缩支持) 命令行工具代替默认的 npm:

$ npm install -g cnpm –registry=https://registry.npm.taobao.org

2、通过添加 npm 参数 alias 一个新命令:

3、安装 Angular 命令行界面工具 —— Angular CLI

npm install -g @angular/cli

注意，如果安装成功之后，输入 ng -v 无响应，估计是安装到了 /home/ubuntu/software/node-v8.9.4-linux-x64/bin 目录下，这个时候，用一个软链接 连接到 /user/local/bin 下就可以了。

4、创建Angular新项目

ng new my-app

5、启动 Angular 新项目

cd my-app

ng serve -o
